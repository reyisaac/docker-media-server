RADARR CUSTOM FORMATS - QUICK IMPORT GUIDE
═══════════════════════════════════════════════════════════════

STEP-BY-STEP:

1. Open Radarr: http://localhost:7878

2. Go to: Settings (top menu) → Show Advanced (toggle top right)

3. Click: Custom Formats (left sidebar)

4. Click: Import button (top right, blue plus icon)

5. For EACH JSON file below, copy the entire content and paste into the import dialog:

───────────────────────────────────────────────────────────────
FILE 1: radarr-custom-format-x265.json
───────────────────────────────────────────────────────────────
This makes Radarr PREFER x265/HEVC releases (+15 points)

Open file: radarr-custom-format-x265.json
Copy entire content, paste into Radarr Import dialog
Click Import

───────────────────────────────────────────────────────────────
FILE 2: radarr-custom-format-good-encoders.json
───────────────────────────────────────────────────────────────
This makes Radarr PREFER quality release groups (+10 points)

Open file: radarr-custom-format-good-encoders.json
Copy entire content, paste into Radarr Import dialog
Click Import

───────────────────────────────────────────────────────────────
FILE 3: radarr-custom-format-avoid-bloated.json
───────────────────────────────────────────────────────────────
This makes Radarr AVOID bloated 1080p releases (-100 points)

Open file: radarr-custom-format-avoid-bloated.json
Copy entire content, paste into Radarr Import dialog
Click Import

───────────────────────────────────────────────────────────────
ASSIGN TO PROFILE:
───────────────────────────────────────────────────────────────

After importing all 3 formats:

1. Go to: Settings → Profiles

2. Click on your profile (HD-1080p or Any)

3. Go to: Custom Formats tab

4. Set these scores:
   • x265/HEVC Preferred: +15
   • Good Encoders: +10
   • Bloated Releases: -100

5. Set: Minimum Custom Format Score = 0

6. Click Save

───────────────────────────────────────────────────────────────
VERIFY IT'S WORKING:
───────────────────────────────────────────────────────────────

1. Go to Movies → Add New Movie

2. Search for any movie

3. Click "Manual Search" (magnifying glass icon)

4. Look at the release scores - you should see:
   • Releases with "x265" or "HEVC" have +15 points
   • Releases from good groups have +10 points
   • Bloated releases (>50GB for 1080p) have -100 points

5. Top-scored release will be auto-selected ✅

═══════════════════════════════════════════════════════════════

Done! Radarr will now automatically prefer efficient releases
and avoid bloated files!
